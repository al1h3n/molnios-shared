# _[MolniOS](https://codeberg.org/al1h3n/molnios-install)_ wezterm keybinds:

### Input
`Ctrl+r` - open fuzzy history to choose.<br>
`Ctrl+u` - clean input.<br>
`Ctrl+Shift+u` - unicode characters selector.<br>
`Ctrl+Shift+C/V`/ `Ctrl+C` - copy/paste.<br>
`Ctrl+Shift+F` - search.<br>

### Actions
`Ctrl+Shift+P` - main action list.<br>
`Ctrl+Shift+L` - debug overlay.<br>

### Windows, tabs
`Ctrl+Shift+N` - new window.<br>

### Size
`Alt+Enter` - fullscreen.<br>
