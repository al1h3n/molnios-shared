Use `"on-scroll-up": "sh po"` to add shell integrations into element. (WORKS ONLY WITH COMMANDS, NOT REAL ACTIONS)

## Important things before usage
1. Change location in `weather.jsonc` to your actual city.
2. To change `hyprland/niri` settings opem window-manager.
3. To change style, open `style.css`, colors - `theme.css`

- [Mechabar](https://github.com/sejjy/mechabar) for modules and file structure.
- [Catpuccin](https://catppuccin.com/) for most of the colors.
