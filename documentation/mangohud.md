# _[MolniOS](https://codeberg.org/al1h3n/molnios-install)_ mangohud keybinds:
Mangohud is a great tool like RTSS on Windows to show FPS.<br>
| Keybind | Definition |
| :--- | :---: |
| `Shift_R+F12` | __Toggle__ HUD. |
| `Shift_R+F11` | Switch HUD position. |
| `Shift_R+F10` | Switch preset of system information. |
| `Shift_R+F1` | Change FPS __limit__. |
| `Shift_L+F2` | Write log (in .csv file, `~/java...`). |
| `Shift_L+F4` | Reload config (by the way it's already real-time). |
| `Shift_R+F9` | Reset FPS metrics. |

### How to enable mangohud in game.
- `prismlauncher`: enable in `Minecraft` section.<br>
- `Steam`: add `MANGOHUD_CONFIGFILE=$L_PATH/config/mangohud-gaming.conf mangohud gamemoderun %COMMAND%` in game launch settings.<br>
