# _[MolniOS](https://codeberg.org/al1h3n/install)_ hyprland keybinds:
`Caps Lock` = Control.<br>
`Alt + Tab` - window switcher, `Alt + Shift + Tab` - window menu switcher (advanced).<br>
`PrintScreen` - make _screenshot_ and save it to __clipboard__, __`~/Pictures/screenshot_<date>.png`__. `Super + PrintScreen` - __record__ screen.<br>
`Super + Shift/Ctrl + PrintScreen` - record screen with _audio_ or via __OBS__.

`Super + Q/W/E/R/T/P` - terminal, __free__ window mode, file manager, run application, run command, private workspace.<br>
`Super + A/S/D/F/J/K/L` -  notes, browser, show desktop full screen, _previous_ song, __pause|unpause__, _next_ song.<br>
`Super + Z/X/C/V` - exit hyprland, emoji picker, close, clipboard manager.

`Super + Escape` - hibernate.<br>
`Super + Shift + Escape` - lock screen.<br>
`Super + Shift + Q/W/E/R/P` - __do not stretch window__, __switch__ window axis, text editor, toggle wallpaper on/off, move to private workspace.<br>
`Super + Shift + A/S/D/F/G/K; H/J/K/L` - open note app, show desktop, gamemode, wallpaper _engine_. Move window to specified location.<br>
`Super + Shift + Z/X/C/V/B/M` - open coder/browser/music player/bluetooth/network manager/media player.<br>
`Super + Shift + F1/2/3/4/5/6` - __volume__ up/down/mute, mute microphone, turn __brightness__ up/down.<br>
`Super + Ctrl + F1/2/3/4/5/6/7/8` - show volume level, time, date, brighness value, GPU/CPU/RAM load, temperature.

`Super + Ctrl + P` - move from private workspace.

## Default keybinds
`Super + x, x[0-9]` - workspaces. `Super + Shift + x, x[0-9]` - move window to particular one.<br>
`Super + arrows` - switch to window.<br>
`Super + Shift + arrow up/down` - zoom/unzoom.

## Default keybinds: mouse
`Super + mouse wheel` - move through workspaces.<br>
`Super + LMB/RMB` - drag/size windows.
