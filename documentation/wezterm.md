# _[MolniOS](https://codeberg.org/al1h3n/molnios-install)_ [wezterm](https://wezterm.org/config/default-keys.html) keybinds:

## Definitions
`Ctrl + Shift` = `Main`

### Input
`Ctrl + R` - open fuzzy history to choose.<br>
`Ctrl + U` - clean input.<br>
`Main + U` - unicode characters selector.<br>
`Main + C/V` | `Ctrl + C` - copy/paste.<br>
`Main + F` - search.<br>
`Main + Space` - select number.<br>

### Actions
`Main + P` - __main action list__.<br>
`Main + L` - debug overlay.<br>

### Windows, tabs
`Main + N` - new window.<br>
`Main + T/W` - create/close tab. `Main + number` - select tab.<br>
`Ctrl/Main + Tab` | `Mouse wheel` while hovering on tab menu - select previous/next tab.<br>
`Main + PgUp/Down` - move tabs.<br>

### Size
`Alt + Enter` - fullscreen.<br>
`Main + minus/=` | `Super + minus/=` - change size of content.<br>
`Ctrl/Super + 0` - reset size.<br>

### Custom binds
`Main + G/B` - create vertical/horizontal internal window.<br>
`Main + W` - close internal window (higher priority than tab).<br>
