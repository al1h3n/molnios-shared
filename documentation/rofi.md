# _[MolniOS](https://codeberg.org/al1h3n/molnios-install)_ rofi keybinds:

### Navigating
`Control + Tab`, `Alt + 1~0` - switch types.<br>
`Alt + .` - shift text if too big.<br>

### Text
Type `-x` to exclude part of a word.<br>
`Ctrl + BackSpace` - remote word.<br>
`Ctrl + Up/Down` - search history.<br>
`Ctrl + Q/W/E` - left text position, clear line, right text position.<br>
`Ctrl + A/S` - remove text on left/right side.<br>

#### Reset config
`rofi -dump-config > ~/.config/rofi/config.rasi`<br>
